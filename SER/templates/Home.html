Speech Emotion Recognition

Speech Emotion Recognition is the task of recognising different emotions from audio file uploaded by the user. For this, user has to first do registration for uploading the file. Then user will upload a .wav audio file which will get store in the backend and then emotion is predicted on the basis of gender.

This can be used further in various application such as Hospital Feedback system, Psychological applications, Mood-synchronizing music players etc.
